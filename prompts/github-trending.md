# GitHub Trending 周报

总结过去一周的 GitHub Trending，包含一句话总结和 Top10 项目即可。

请同时将内容写入 Obsidian 的 Digest 文件夹，路径为：

```text
~/Obsidian/Main/Digest
```

文件命名格式为：

```text
W[周序号]-Github-Trending.md
```

例如：

```text
W04-Github-Trending.md
```

不要只告诉我文件已写入；即使已经写入 Obsidian，也请在对话里完整展示总结内容。

展示内容不要按 Markdown 代码块的形式，正常输出即可。
